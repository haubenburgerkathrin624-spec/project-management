# 信息化项目预算与流程管理

本仓库为信息化项目管理 / 信息化建设项目预算与流程管理的整合项目资料，包含前端页面重设计原型、需求分析、PRD、详细设计方案、旧系统手册与 UI 参考图。

## 原型入口

打开：

`信息化项目预算与流程管理/前端页面重设计原型/index.html`

该原型为静态 HTML/CSS/JS 页面，可直接在浏览器中打开继续查看和修改。

## 主要目录

- `信息化项目预算与流程管理/前端页面重设计原型/`：已修改的前端原型与素材。
- `信息化项目预算与流程管理/改造方案输出/`：PRD 与详细设计方案。
- `信息化项目预算与流程管理/转写结果/`：录音转写结果与分段音频。
- `ui设计图/`：页面设计参考图。
- `上海电机学院项目管理系统-用户手册.docx`：旧系统用户手册参考。
